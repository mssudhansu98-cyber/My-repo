# Getting started
## Prerequisites
- Smartphone(Android or iOS)
- Internet connection
- Valid email ID and phone number

## Install the Application

1. Open Google Play Store or App Store
2. Search for "MakeMy Trip"
3. Tap **Install**
4. Open the application after installation

## Sign In/Sign Up
1. Open the app
2. Tap **Login / Sign Up**
3. Enter your mobile number or email
4. Verify using OTP

## Result
You can now access all the features of the application.
