# Book a Flight Ticket


## Prerequisites
- Logged - in user
- Selected flight

## Steps to Book a ticket
1. Select a flight from search results
2. Tap ***Book Now**
3. Enter message details:
 - Name
 - Age
 - Contact information
4. Tap **Contninue**
5. Select payment method
6. Complete Payment

   ## Result
   Ticket is successfully booked.

   ## Confirmation
- Ticket details will be sent via mail and SMS
