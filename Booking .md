# Book a Flight Ticket

## Prerequisite
- Logged-in user
- Selected flight

## Steps to Book a Ticket
1. Select a flight from search results
2. Tap **Book Now**
3. Enter passenger details:
   - Name
   - Age
   - Contact information
4. Tap **Continue**
5. Select payment method
6. Complete payment

## Result
Ticket is successfully booked.

## Confirmation
- Ticket details will be sent via email and SMS

## Notes
- Verify details before payment
- Ensure sufficient balance for payment
